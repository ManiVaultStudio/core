# Data Hierarchy Plugin

View plugin for interacting with the ManiVault data hierarchy.

<img width="322" height="149" alt="image" src="https://github.com/user-attachments/assets/4fbdbe32-44be-4218-9ed7-44ff7434b257" />

- Lists all data sets
- Derived data is listed indented below its parent
- Icons indicate data types
- The context menu (right-click) shows:
  - Import options for new data
  - Analysis and view options for clicked data
  - Many other options like locking or deletion
- Progress for asynchronous compute tasks is shown
